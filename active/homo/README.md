# 恶臭数字论证器
一个将任意数字分解成 114514 构成的公式的工具。

## 项目首页
[https://lab.magiconch.com/homo/](https://lab.magiconch.com/homo/)

## 引用方式
```HTML
<script src="homo.js"></script>
<script>
let 恶臭 = homo(1919810);
// "114514*(11-4-5+14)+(114*514+(114*51*4+(1145*(1+4)+(11-4+5+1-4))))"
</script>
```
